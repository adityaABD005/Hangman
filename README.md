# Hangman Game (GUI) — Resume Project Description

## Project Summary

Developed a desktop-based Hangman game using Python and Tkinter, featuring an interactive graphical interface and dynamic game logic. The application allows users to guess letters, track attempts, and visualize progress through a real-time drawing of the hangman.

---

## Key Contributions

* Designed and implemented a GUI-based game using Tkinter for interactive gameplay
* Built core game logic including word selection, letter guessing, and win/lose conditions
* Implemented real-time visual feedback using Canvas to draw the hangman step-by-step
* Added input validation and user feedback for invalid or repeated guesses
* Integrated keyboard event handling (Enter key binding) for improved user experience
* Developed restart functionality to reset game state dynamically

---

## Features

* Random word generation from a predefined dataset
* Live word display with hidden letters (_ _ _ format)
* Attempt tracking system with maximum limit
* Visual hangman drawing based on incorrect guesses
* Restart game option
* User-friendly interface with real-time updates

---

## Tech Stack

* Python
* Tkinter (GUI Development)
* Canvas (for graphics rendering)
* Random module (word selection logic)
